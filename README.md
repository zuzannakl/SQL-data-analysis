# SQL Data Analysis

This repository contains SQL projects focused on data analysis, querying relational databases, and extracting information from real-world datasets.

## Project Structure

### data-retrieval-and-filtering.sql
Advanced data querying techniques:
- filtering and sorting data
- pattern matching using `LIKE`
- subqueries
- pagination with `LIMIT` and `OFFSET`

### relational-data-and-joins.sql
Working with relational databases:
- multi-table `JOIN` operations
- aggregations and grouping
- time calculations with `TIMESTAMPDIFF`

### database-management.sql
Database management and data manipulation:
- table creation
- data updates and migrations
- data cleanup operations
